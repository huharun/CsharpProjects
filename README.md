## **Mini Projects Dashboard**  

This is a fun console-based dashboard where you can explore **five interactive C# mini-projects**. Each one demonstrates core programming concepts like loops, conditionals, and data structures.  

**Projects included:**  

1. **Calculator** – Basic arithmetic operations with input validation.  
2. **ToDo App** – Add, view, and remove tasks in a simple task manager.  
3. **Guess Number** – Number guessing game with attempt tracking.  
4. **Quiz Game** – Small multiple-choice quiz with scoring.  
5. **Notes App** – Add, view, and delete notes quickly.
